Este programa/libreria te permite obtener el href y texto de los links de archivos markdown y analizar los links únicos, rotos y totales.

1. INICIO
2. Lee path, options
3. SI path = absolute
    ENTONCES Leer path
    SI NO
        SI path = relative
            ENTONCES Function (Convertir path = absolute)
                     Leer path 
        SI NO 
            Imprimir "Ingresar una ruta válida"
  - FIN de condición -
4. SI path = file 
    ENTONCES SI file = .md
        ENTONCES Función 
                          (recorrer links de file
                          leer links
                          obtener su href y text 
                          almacenar href y text en un array de objetos)
                SI text.length > 50
                    ENTONCES 
                    Función 
                           (Limitar text.length = 50)
      
   SI NO
4.1 SI path = dir
      ENTONCES Función
                      (Leer dir
                       Aplicar recursión a dir
                       Ejecutar caso base: Find file.md)
      
    SI NO 
      Imprimir "Ingresar una ruta válida"
  - FIN de condición -

5. SI options = null || ''
    ENTONCES Imprimir path, href y text.
5.1     SI NO --validate
        ENTONCES función Validate 
            (validar Si links de array = OK SI NO FAIL y almacenar en array) 
5.1.1 Imprimir path, href, texto, OK o FAIL
5.2     SI NO --stats
        ENTONCES función StatsTotalandUnique 
            (calcular links totales y unicos de array y almacenar en array)
5.2.1 Imprimir path y total y unique con su valor según resultado de funcion                5.3     SI NO --validate y --stats
        ENTONCES 
            función optionVal 
                (calcular links broken ejecutando función Validate obteniendo solo FAIL y almacenar en array)
            función optionStats 
                (calcular links totales y unicos ejecutando la función StatsTotalandUnique y almacenar en array )
  - FIN de condición -
5.3.1 Imprimir path, total, unique y broken con su valor númerico según resultado de funcion
7.Fin
