Este programa/libreria te permite obtener el href y texto de los links de archivos markdown , analizar los links únicos, rotos, totales o aquellos que funcionan o no.

1. INICIO
2. Usuario ejecuta modulo con sus argumentos
3. LEER el primer argumento que es ruta
4. SI ruta es absoluta 
    ENTONCES
    Leer el contenido de la ruta
   SI NO es absoluta
      Validar Si ruta es relativa
        ENTONCES
        Convertir ruta a absoluta y leer contenido de la ruta
      SI NO es ni relativa ni absoluta
        Imprimir mensaje para ingresar una ruta válida
   FIN de condición 
4.1 SI ruta es un archivo
    ENTONCES 
    Validar SI archivo es markdown
      ENTONCES
      Ejecutar función para buscar y leer los links del archivo, obteniendo su href y texto para luego almacenar ambos datos en un array de objetos
      SI texto mayor a 50 caracteres 
        ENTONCES Limitar texto a 50 caracteres
    SI NO es markdown
      Imprimir mensaje para ingresar archivo markdown
   SI la ruta NO es un archivo
    Validar SI es un directorio
      ENTONCES
      Aplicar función para leer directorio, aplicandole recursión y ejecutar caso base: Encontrar archivo markdown
      SI NO es directorio
        Imprimir mensaje para ingresar una ruta válida
5. SI segundo argumento no existe
    Imprimir la ruta, el href y texto truncado a 50 caracteres
    SI NO segundo argumento si existe
5.1     Validar SI segundo argumento es --validate
          ENTONCES Ejecutar función --validate para validar si los links almacenados en array funcionan asignandoles un OK si no FAIL y almacenar este nuevo dato en el array como nuevo dato llamado val con el valor OK o FAIL
5.1.1   Imprimir ruta, href, texto y val es OK o FAIL
5.2     DE LO CONTRARIO segundo argumento es --stats
          ENTONCES Ejecutar función --stats para calcular links totales y únicos en el/los archivo(s) y almacenar links en el array de objetos como nuevo dato llamado total y unique que contiene un valor numerico
5.2.1   Imprimir links totales y únicos con un valor númerico
5.3     SI NO segundo argumento es --validate y --stats
          ENTONCES Calcular links rotos ejecutando la función --validate, sacando solo los links que son FAIL y almacenarlos en array de objetos como nuevo dato llamado broken que contendra un valor numerico
          Ejecutar la función --stats para obtener los links totales y unicos, luego almacenarlos en array de objetos como dato llamado total y unique
5.3.1   Imprimir links totales, únicos y broken con un valor númerico
6. Fin
